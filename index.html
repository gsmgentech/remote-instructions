Customer Remote USB Instructions

Overview:
This service uses USB Redirector – Technician Edition (Customer Module) to securely provide temporary remote access to your USB device. You will connect directly to a certified technician using a unique Technician ID. Administrator privileges are required.

Prerequisites

  • Stable internet connection
  • USB device and appropriate data cable
  • Windows-based computer or laptop

Connection Procedure

  1. Download and install USB Redirector (Customer Module).
  2. Launch the application using Run as Administrator.
  3. In the Connect With Technician window:
    • Enter the provided Technician ID.
    • Enter your name in the Additional Information field.
    • Select Connect to begin the session.
  4. Connect the USB device to be shared. Reconnect if already plugged in.
  5. Remain connected while the technician completes the service.

Security Assurance: Only the selected USB device is accessible during the session. No other files, applications, or personal data are exposed.

Troubleshooting

  • Connection Failure: Verify Technician ID and internet connectivity.
  • USB Not Detected: Restart the application as Administrator.
  • Security Warning: Proceed only if downloaded from the official source.
